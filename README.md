# Simple Products API Project

A simple React application that fetches product data from the DummyJSON API and displays it in a responsive product grid with pagination. This project was created to practice React fundamentals such as API integration, component-based architecture, state management, routing, and pagination.

## Features

* Fetches products from the DummyJSON API
* Displays products in responsive cards
* Pagination using Previous and Next buttons
* Product details page using React Router
* Loading state while fetching data
* Responsive layout using CSS Grid

## Technologies Used

* React
* React Router DOM
* Axios
* CSS3
* DummyJSON API

## Installation

1. Clone the repository

```bash
git clone https://github.com/sumedh22kar/Simple_ProductsApi_project.git
```

2. Navigate to the project folder

```bash
cd Simple_ProductsApi_project
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
